传感器基础课程
===============
通过传感器基础课程你可学会如果通过Arduino IDE编程控制Pico开发板连接的模块以及读取传感器数据等操作。

特别注意：实验时，模块/传感器连接线材时，必须按照资料里的接线方法及位置，电源与信息脚不能错接，否则会损坏模块/传感器。
